**Lab5_FX: Telegram-бот для обработки текста**

Данный проект является пятой лабораторной работой, посвященной изучению архитектуры клиент-серверного взаимодействия и работы с Telegram Bot API.
Программа представляет собой бота, который принимает текстовые сообщения и обрабатывает их согласно заданному алгоритму.

**Функциональные возможности**

	•	Механизм взаимодействия: использование технологии Long Polling для опроса серверов Telegram на наличие новых обновлений
	•	Обработка сообщений: прием текста от пользователя, выполнение логических операций в классе TextProcessor и отправка результата обратно в чат
	•	Индивидуальный алгоритм: реализация конкретной задачи по обработке строк (например, шифрование, фильтрация или анализ текста) согласно варианту

**Архитектура проекта**

Приложение имеет четкую структуру для разделения ответственности:

	•	BotConfig: класс для хранения конфигурационных данных, включая уникальный API-токен и имя бота
	•	TextProcessorBot: основной класс логики бота, реализующий интерфейс потребителя обновлений (LongPollingSingleThreadUpdateConsumer)
	•	TextProcessor: компонент, содержащий реализацию алгоритма обработки текста
	•	TextBotApplication: главный класс (точка входа), отвечающий за инициализацию клиента и регистрацию бота
